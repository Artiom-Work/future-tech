# $\color{orange}\textsf{Future Tech}$

$\color{limegreen}\text{Учебная работа}$

## $\color{mediumblue}\text{Описание работы }$:

Внешняя часть информационно-образовательного портала об искусственном интеллекте

Работа на основе практического видеоурока. Автор - [**Александр Ламков**](https://www.youtube.com/watch?v=hkYzqTKnSIg&t=25495s).

**Цели и задачи работы :**

❗Расширенное применение БЭМ-методологии в вёрстке.

❗Применение глубокого и продуманного анализа макета сайта на этапе вёрстки.

❗Практика компонентного подхода в вёрстке.

❗Формирование файловой структуры сайта на основе компонентного подхода.

❗Практика широкого применения Sass миксинов, функций и медиазапросов.

❗Практика широкого применения глобальных и локальных CSS переменных.

❗Практика широкого применения утилитарных селекторов классов.

❗Применение rem единиц измерения на всём сайте.

❗Написание Js модулей для отдельных компонентов сайта с использованием Js-классов.

🎯 $\color{mediumblue}\textsf{Основная задача}$ - Изучение профессионального подхода в веб-разработке на этапе вёрстки.

## $\color{mediumblue}\text{Новые (для меня) способы вёрстки и используемые технологии:}$

💡 Удобный набор SCSS миксинов и функций для работы со стилями.

💡 Удобство работы с JS-классами. Создание переиспользуемых компонентов.

💡 Способ создания компонента «Аккордеон» по всем правилам доступности и без использования JS.

💡 Способ создания кастомного компонента select по всем правилам доступности, а также возможность расширения функционала с помощью JS.

💡 Способ создания компонента expandableContent (расширяемый контент) с использованием JS.

💡 Способ создания компонента tab-панели по всем правилам доступности с использованием JS.

💡 Способ стилизации кастомной видеопанели тега video.

💡 Использование библиотеки IMask в комбинации с кастомным JS для создания маски номера телефона на поле ввода данных пользователем.

💡 Знакомство с понятием «абстрактного JS-класса».

💡 Расширенное использование и углубленное понимание на практике JS MatchMedia API.

💡 Удобный компилятор SCSS и установка его через npm.

💡 Удобство работы с JS с помощью data-атрибутов.

💡 Практическое применение CSS-локальных переменных и их возможностей для унификации в рамках компонента.

💡 И многое другое...
